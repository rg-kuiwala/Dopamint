# 🎨 Dopamint

> Tap. Relax. Feel Better.

A minimal, satisfying color game designed to help you relax, reduce stress, and feel better in seconds.

🌐 Live Demo: https://dopamint.vercel.app

---

## 🧠 About

Dopamint is a lightweight, relaxing web app focused on simple interactions and smooth color transitions.  
It creates a calming experience that helps reduce stress, improve focus, and deliver quick mental refresh.

---

## ✨ Features

- 🎨 Smooth and satisfying color transitions  
- 🧠 Designed for stress relief and relaxation  
- ⚡ Quick 30–60 second calming sessions  
- 🔁 Simple, repeatable interaction loop  
- 📱 Minimal and clean user interface  

---

## 🌈 Use Cases

- Stress relief after work  
- Quick mental refresh during breaks  
- Reducing anxiety and overthinking  
- Relaxing before sleep  

---

## 🚀 Tech Stack

- HTML  
- CSS  
- JavaScript  
- PWA (Progressive Web App)  
- Vercel Deployment
